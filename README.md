# 🌈 초등학생용 MBTI 성격 검사

재미있는 질문으로 알아보는 나의 성격 유형!

## 📖 소개

초등학생들이 쉽고 재미있게 자신의 성격 유형을 알아볼 수 있는 MBTI 검사 웹사이트입니다.

### ✨ 주요 특징

- 🎮 **쉽고 재미있어요**: 초등학생 눈높이에 맞춘 질문
- 🌈 **16가지 유형**: 각 유형별 귀여운 캐릭터
- 📱 **모바일 친화적**: 스마트폰에서도 편하게 이용 가능
- 🎨 **예쁜 디자인**: 밝고 친근한 색상과 애니메이션

## 🚀 시작하기

### 온라인에서 바로 사용

GitHub Pages를 통해 배포된 사이트에 접속하세요:
👉 [https://sungminko-smko.github.io/mbti-test/](https://sungminko-smko.github.io/mbti-test/)

### 로컬에서 실행

```bash
# 저장소 클론
git clone https://github.com/SungminKo-smko/mbti-test.git

# 폴더로 이동
cd mbti-test

# index.html 파일을 브라우저로 열기
```

## 📁 프로젝트 구조

```
mbti-test/
├── index.html              # 메인 페이지
├── test.html               # 검사 페이지
├── result.html             # 결과 페이지
├── css/
│   ├── style.css           # 메인 스타일
│   ├── test.css            # 검사 페이지 스타일
│   └── result.css          # 결과 페이지 스타일
├── js/
│   ├── questions.js        # 질문 데이터 (16문항)
│   ├── test.js             # 검사 로직
│   └── result.js           # 결과 표시 로직
├── data/
│   └── types.json          # MBTI 16가지 유형 설명
└── README.md
```

## 🎯 사용 방법

1. **시작하기**: 메인 페이지에서 "검사 시작하기" 버튼 클릭
2. **질문 답하기**: 16개의 질문에 마음 가는 대로 답변
3. **결과 확인**: 나의 MBTI 유형과 특징 확인
4. **공유하기**: 친구들에게 결과 공유

## 🛠 기술 스택

- **HTML5**: 웹 페이지 구조
- **CSS3**: 스타일링 및 애니메이션
- **JavaScript (Vanilla)**: 로직 구현
- **GitHub Pages**: 무료 호스팅

## 📋 검사 구성

### 질문 구성
- **E vs I (외향 vs 내향)**: 4문항
- **S vs N (감각 vs 직관)**: 4문항
- **T vs F (사고 vs 감정)**: 4문항
- **J vs P (판단 vs 인식)**: 4문항

총 16문항으로 약 3분 소요

### MBTI 16가지 유형
각 유형별로 귀여운 동물 캐릭터와 특징 설명 제공

| 유형 | 캐릭터 | 별명 |
|------|--------|------|
| ISTJ | 🦫 비버 | 성실한 관리자 |
| ISFJ | 🐨 코알라 | 따뜻한 수호자 |
| INFJ | 🦉 부엉이 | 신비로운 예언자 |
| INTJ | 🦅 독수리 | 전략가 |
| ISTP | 🐱 고양이 | 만능 재주꾼 |
| ISFP | 🐰 토끼 | 예술가 |
| INFP | 🦄 유니콘 | 꿈꾸는 이상주의자 |
| INTP | 🦊 여우 | 논리적인 사색가 |
| ESTP | 🐯 호랑이 | 모험가 |
| ESFP | 🦩 플라밍고 | 자유로운 연예인 |
| ENFP | 🐬 돌고래 | 재기발랄한 활동가 |
| ENTP | 🦜 앵무새 | 발명가 |
| ESTJ | 🦁 사자 | 엄격한 관리자 |
| ESFJ | 🐶 강아지 | 친선도모자 |
| ENFJ | 🦒 기린 | 정의로운 리더 |
| ENTJ | 🦖 티라노 | 대담한 지휘관 |

## ⚠️ 주의사항

- 이 검사는 **재미를 위한 것**이며, 전문적인 심리 검사가 아닙니다
- 실제 MBTI 검사와는 다를 수 있습니다
